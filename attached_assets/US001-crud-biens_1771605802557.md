# US001 - CRUD Biens

## User Story

**En tant que** email-parser ou frontend  
**Je veux** gerer les biens via API REST  
**Afin de** creer, lire, mettre a jour les proprietes

---

## Criteres d'acceptation

### Scenario 1: Lister les biens
**Donne** une base avec 50 biens  
**Quand** GET /api/biens?page=1&limit=20  
**Alors** les 20 premiers biens sont retournes avec pagination

### Scenario 2: Creer un bien
**Donne** des donnees valides  
**Quand** POST /api/biens  
**Alors** le bien est cree et retourne avec id

### Scenario 3: Validation
**Donne** des donnees sans adresse  
**Quand** POST /api/biens  
**Alors** erreur 400 avec details des champs manquants

### Scenario 4: Rechercher un bien
**Donne** une adresse partielle  
**Quand** GET /api/biens/search?adresse=ECONDEAUX&code_postal=93800  
**Alors** les biens correspondants sont retournes avec score

## Schema base de donnees

```sql
CREATE TABLE biens (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    adresse TEXT NOT NULL,
    complement_adresse TEXT,
    code_postal TEXT NOT NULL,
    ville TEXT NOT NULL,
    gestionnaire_id INTEGER NOT NULL,
    information TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (gestionnaire_id) REFERENCES gestionnaires(id)
);

CREATE INDEX idx_biens_code_postal ON biens(code_postal);
CREATE INDEX idx_biens_adresse ON biens(adresse);
```

## Algorithme de recherche

```python
def search_bien(adresse: str, code_postal: str) -> List[BienMatch]:
    # 1. Normaliser l'adresse
    normalized = normalize_address(adresse)
    
    # 2. Rechercher par code postal exact
    candidates = Bien.query.filter_by(code_postal=code_postal).all()
    
    # 3. Scorer chaque candidat
    matches = []
    for bien in candidates:
        score = calculate_similarity(normalized, normalize_address(bien.adresse))
        if score > 0.5:
            matches.append(BienMatch(bien, score))
    
    # 4. Trier par score
    matches.sort(key=lambda x: x.score, reverse=True)
    
    return matches
```
